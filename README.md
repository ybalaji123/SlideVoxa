# SlideVoxa
**About:**
SlideVoxa is an AI-powered web application designed to transform traditional PowerPoint presentations into engaging, narrated presentations automatically. Many presenters spend significant time preparing scripts and practicing narration for their slides. SlideVoxa simplifies this process by automatically generating presentation scripts and natural-sounding voiceovers directly from slide content.

The system analyzes the structure and text of a PowerPoint (.pptx) file and uses Artificial Intelligence to generate a clear, presentation-style narration for each slide. The generated script is then converted into high-quality speech using text-to-speech models, allowing users to experience their presentation as a fully narrated slideshow.

SlideVoxa also includes a web-based presentation player that synchronizes slides with generated audio narration, creating a smooth and professional presentation experience. Additionally, the platform can generate potential audience questions based on the presentation content, helping presenters prepare for discussions and Q&A sessions.

The platform uses technologies such as FastAPI for backend services, MongoDB for data storage, and AI models for script generation and voice synthesis. By automating narration and presentation preparation, SlideVoxa aims to make presentations more interactive, accessible, and efficient for students, educators, and professionals.


# Key Features
* Automated script generation from slides
* AI voice narration (realistic speech)
* High-quality slide rendering
* Interactive Q&A generation
* Web-based presentation player

# Tech Stack
* **Backend:** FastAPI (Python)
* **Database:** MongoDB, GridFS
* **Frontend:** HTML, CSS, JavaScript
* **AI:** Sarvam AI (LLM), ElevenLabs (TTS)
